The OWL Metrics Project

OWLMetrics is a JAVA project for generating metrics for ontologies. It has been implemented using the OWL API. In particular, it extends the package OWLMetrics from the OWLAPI.

Documentation

A description of the implemented metrics and their usage is described in the published paper:

  • E. Mikroyannidi, R. Stevens, and A. Rector. Identifying ontology design styles with metrics. In 7th International Workshop on Semantic Web Enabled Software Engineering (SWESE), 10th International Semantic Web Conference (ISWC) 2011.[PDF]
